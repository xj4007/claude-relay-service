# Feature: Home Page

## Goal
Display a nine-grid menu for quick access to features.

## Details
- **Style**: Minimalist Black & White.
- **Layout**: 3x3 Grid.
- **Items**:
    1. Check-in (签到)
    2. Balance (余额)
    3. Logs (日志)
    4. Recharge (充值)
    5. More (更多)
    6. Settings (设置)
    7. Help (帮助)
    8. About (关于)
- **State**:
    - If no key selected: Show "Configure Key" prompt.
    - If key selected: Show Grid + Current Key Info.
